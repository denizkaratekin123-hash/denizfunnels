# Deniz Funnels - VSL Landing Page

A high-converting VSL (Video Sales Letter) landing page for Deniz Funnels agency services.

## Features

- Premium dark theme with cyan/blue accent colors
- Embedded Vimeo VSL with email gate
- Cal.com calendar integration
- Scroll-triggered animations
- Mobile-responsive design
- Video testimonial carousels
- Interactive method/process section
- FAQ accordion

## Tech Stack

- Pure HTML/CSS/JavaScript (no frameworks)
- Vimeo Player API
- Cal.com embed
- Intersection Observer for scroll animations

## Structure

```
/
├── index.html          # Main landing page
└── Assets/
    ├── logo.png        # Deniz Funnels logo
    ├── pagani.jpg      # Lifestyle proof image
    └── corner.png      # CTA section accent image
```

## Usage

Simply open `index.html` in a browser or deploy to any static hosting service (GitHub Pages, Netlify, Vercel, etc.).
